# 项目简介

## 核心功能
- 基于YOLOv8的棉花叶病检测，支持6类疾病识别（blight, curl, grey mildew, healthy, leaf spot, wilt）
- 提供完整的源代码、数据集（4000张标注图片）、训练好的模型及训练结果
- 支持多种输入方式：单张图片、图片文件夹、视频文件、本地摄像头
- 提供实时检测结果，包括目标位置、目标总数和置信度信息
- 支持自定义检测参数，如置信度阈值和IOU阈值
- 支持批量检测并查看所有结果
- 提供UI界面，可选择显示标签和原图

## 适用场景
- 农业科研机构进行棉花病害研究
- 农业技术推广部门进行病害监测
- 棉花种植户进行田间病害自检
- 农业院校教学演示

## 技术特点
- 基于Python3.9开发，使用Pycharm/VSCode+Anaconda环境
- 主要依赖库包括opencv-python、PyQt5、ultralytics、torch1.9等
- 采用YOLOv8深度学习模型进行目标检测
- 提供完整的用户界面，操作简便
- 包含详细的项目运行教程和使用说明
